## 制作本地文档

### Html 文档

主要步骤为：

* 克隆本项目
* 安装 sphinx
* 生成文档（文档生成在 _bulid 目录下)

命令示例：

```shell
git clone git@github.com:dormouse/Flask_Docs_ZhCn.git
cd Flask_Docs_ZhCn/
conda create -n flask_doc python=3*
source activate flask_doc
pip install -r requirements.txt
make html
```
